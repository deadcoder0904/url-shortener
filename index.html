URL Shortener Microservice

Example creation usage:

POST https://url-shortener-0904.herokuapp.com/new?url=https://www.google.com
POST https://url-shortener-0904.herokuapp.com/new?url=http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"https://url-shortener-0904.herokuapp.com/1" }

Usage:

GET https://url-shortener-0904.herokuapp.com/2

Will redirect to:

https://www.google.com/
